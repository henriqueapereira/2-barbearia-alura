  Criar formulários
  Campos para celulares
  Tabelas
  Hierarquia CSS
  Transformações e transições
  Importar conteúdo externo, como fontes, vídeos e mapas
  Pseudo-classes e pseudo-elementos
  Seletores de CSS avançados
  Opacidade e sombra
  Design responsivo e viewport
